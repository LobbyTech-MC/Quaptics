## 量子光学是什么？
Quaptics（量子光学）是一个 [Slimefun](https://github.com/Slimefun/Slimefun4) 附属插件，提供了一些启发智力的东西。玩家可以创建、操纵量子光束，并在一个全新的系统中使用，这并不像电路回路。
- 需要你真的*动脑子*来到后期
- 显示实体提供的无限可能性
- 包含了例如炮塔、多方块点击装置、信标等新机器
- 复杂的多方块系统
